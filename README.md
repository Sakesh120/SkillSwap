# 🔁 SkillSwap – Skill Sharing Platform

SkillSwap is a web-based platform that enables users to exchange skills with each other. Instead of paying for courses, users can **teach what they know and learn what they need** through a collaborative, two-way learning system.

---

## 🚀 Problem Statement

Most online learning platforms follow a **one-way model**:

- Instructor → Student
- Passive content consumption
- Expensive subscriptions

This creates:

- Limited interaction
- Low engagement
- Accessibility issues for many learners

---

## 💡 Solution

SkillSwap introduces a **peer-to-peer learning ecosystem** where:

- Every user can be both a **teacher and a learner**
- Learning is **interactive and practical**
- Skills are exchanged instead of money

---

## 🎯 Features

### 👤 User Authentication

- User registration and login
- Secure authentication system
- Profile creation

### 🧑‍💼 User Profile

- Add skills you can **teach**
- Add skills you want to **learn**
- View other user profiles

### 🔍 Skill Matching

- Find users based on skills
- Connect with people having complementary skills

### 🤝 Request System

- Send skill exchange requests
- Accept or reject requests
- Track request status

### 📅 Session Scheduling

- Schedule sessions between users
- Manage session details

### 📊 Dashboard

- Overview of:
  - Requests
  - Sessions
  - User activity
